# spring7-playground - Projektübersicht

Status: Experimental

Dieses Repository dient als Spielplatz für die Erkundung von Spring 7 und Spring Boot 4. Es enthält verschiedene Beispielprojekte und Module, die die Integration und Nutzung neuer Funktionen und Architekturen demonstrieren.

## Hauptmerkmale:

*   **Spring 7 & Spring Boot 4:** Experimente mit den neuesten Versionen der Spring-Frameworks.
*   **Modulith-Demo:** Ein Beispiel für eine modulare Monolith-Architektur.
*   **KI-Agenten & Assistenten:** Experimente mit verschiedenen KI-bezogenen Komponenten (ai-agent, ai-assistant, ai-gemini, ai-plan, ai-rag).
*   **Maven-basiertes Projekt:** Verwendung von Maven für das Build-Management.
*   **Spring Shell:** Eine Kommandozeilenanwendung, die auf Spring Shell basiert.

## Erste Schritte:

Um das Projekt zu starten, klonen Sie das Repository und führen Sie die Maven-Build-Befehle aus. Weitere Informationen zu den einzelnen Modulen finden Sie in den jeweiligen Unterverzeichnissen.
